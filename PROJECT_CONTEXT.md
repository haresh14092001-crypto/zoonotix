# Project Context

This repository contains ZoonotiX, an educational encyclopedia for veterinary zoonotic diseases. The project is intended to support students, interns, faculty, and public-health educators with a professional, mobile-friendly reference experience.

## Scope

This sprint focuses on repository hardening, documentation, maintainer readiness, and contributor experience. It does not change the existing veterinary content or application behavior.

## Constraints

- Preserve existing functionality.
- Preserve educational content.
- Improve maintainability and collaboration.
- Prepare the repository for future public release.
