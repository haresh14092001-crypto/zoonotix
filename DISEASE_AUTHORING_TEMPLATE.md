# Disease Authoring Template

## ZoonotiX Veterinary Zoonotic Disease Encyclopedia

Version 1.0

---

# Purpose

This document defines the official writing standard for every disease included in ZoonotiX.

Every disease page must follow this template.

No disease should deviate from this structure unless there is a scientifically valid reason.

The objective is to create a consistent, authoritative veterinary encyclopedia suitable for undergraduate BVSc students and future expansion.

---

# Core Principles

Every disease page must be

✔ Scientifically accurate

✔ Educational

✔ Clinically relevant

✔ Mobile friendly

✔ Easy to navigate

✔ Consistent

✔ Visually structured

✔ Encyclopedia quality

---

# Non-Negotiable Rules

Never delete previously approved content.

Never simplify scientific concepts.

Never reduce educational depth.

Never duplicate information.

Never fabricate facts.

Never invent epidemiological statistics.

Never invent treatment protocols.

Never invent references.

Always preserve consistency across diseases.
