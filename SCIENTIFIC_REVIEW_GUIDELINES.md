# SCIENTIFIC_REVIEW_GUIDELINES.md

# ZoonotiX Scientific Review Guidelines

Version: 1.0

Department of Veterinary Public Health & Epidemiology

---

# Purpose

This document defines the scientific quality standards for all educational content within ZoonotiX.

Every disease page, diagram, table, workflow, algorithm, and educational element must comply with these guidelines before being considered complete.

This document exists to ensure consistency, scientific accuracy, academic integrity, and educational value.

---

# Scientific Philosophy

ZoonotiX is an educational encyclopedia.

It is NOT

❌ A blog

❌ A coaching website

❌ Wikipedia

❌ AI generated notes

❌ Medical advice

It is a structured veterinary educational resource intended for undergraduate veterinary students.

---

# Core Principles

Every statement must satisfy four conditions

✓ Scientifically accurate

✓ Educationally useful

✓ Clinically relevant

✓ Evidence-based

If any statement fails one of these conditions, it should be revised or removed.

---

# Accuracy Policy

Never fabricate

• Facts

• Statistics

• Disease prevalence

• Mortality rates

• Drug dosages

• Vaccination schedules

• Government programmes

• Diagnostic accuracy

• Research findings

• References

If evidence is unavailable or conflicting, explicitly state that.

# Terminology Standards

Always use internationally accepted veterinary terminology.

Example

Use

"Brucella abortus"

NOT

"Brucella bacteria"

Use

"Coxiella burnetii"

NOT

"Q Fever bacteria"

Use complete scientific names on first mention.

Subsequent mentions may use abbreviated forms where appropriate.

# Treatment Standards

Treatment information should

Describe principles.

Discuss commonly accepted therapies.

Mention important considerations.

Avoid

Detailed prescription writing.

Exact dosages unless universally accepted.

Off-label recommendations.

Experimental therapies without clear evidence.

# Publication Standard

A page is considered complete only if

• It meets the Disease Authoring Template.

• It complies with these Scientific Review Guidelines.

• It preserves all previous approved content.

• It is suitable for recommendation by a Veterinary Public Health faculty member.

• It provides sufficient depth for BVSc undergraduate students.

• It remains understandable, well-structured, and evidence-based.

# Long-Term Vision

ZoonotiX should become a living veterinary encyclopedia that evolves with advances in veterinary medicine while maintaining the highest standards of scientific accuracy, educational quality, and software engineering.

Every future contribution should move the project closer to becoming the definitive digital reference for veterinary zoonotic diseases.
